# Quantum Information & Computation with Qiskit

IBM Quantum Learning with Qiskit — notebooks covering quantum information basics, quantum circuits, entanglement, and fundamental quantum algorithms.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Contents

- **Basics of quantum information/**
  - Single system (Vectors and Matrices)
  - Multiple systems
  - Quantum Circuits
  - Entanglement in action

## Requirements

- Python 3.11+
- See [requirements.txt](requirements.txt) for full list.
